70. Types of Error:
1. SyntaxError
Definition: Raised when Python detects invalid syntax in the code.

2. ValueError
Definition: Raised when a function receives an argument of the correct type but with an inappropriate value.
Eg:
   int("abc")
Explanation:
The int() function expects a string that represents a valid integer (e.g., "123"). 
The string "abc" is not a valid integer, so Python raises a ValueError.
Thus,int("123") this is correct.

3. ZeroDivisionError
Definition: Raised when attempting to divide a number by zero.

4. FileNotFoundError
Definition: Raised when attempting to access a file or directory that does not exist.

82.Model Evaluation is the process of accessing how well an ML model
performs on unseen data.
It helps to determine the accuracy and the reliability of a model
performance.
It helps to identify issues of overfitting,underfitting and bias.
           MODEL EVALUATION METRICS
A. Accuracy :  This measures the correctness of a prediction but is misleading
for imbalanced datasets.Exanple is where 90% of a dataset belongs to one class,
a model that needs to make a prediction,will predict towards that of the 
90 percent class.
Thus,accuracy is best used for balanced datasets.So 50% males and 
50% females is a balanced dataset.

B. Precision : This measures the level of correctness of a positive prediction
where this is very important when a false positive is costly.
Thus,how accurate is the ones that the model predicted as positive
and it actually is positive.
Eg :
  Example:
Spam filter: marking a real email as spam = bad
            You want high precision

Recall is used when we cannot leave room for making wrong positive corrections.
That’s actually precision, not recall.

C.Recall Recall is how many actual positives were correctly identified.
Recall is considered when we cannot leave room for making wrong positive
corrections thus,in medical diagnostics,making a correct prediction
for cancer is imperative hence recall is used as the main
metric, since missing a correct prediction can be deadly.
In summary,
  Precision	- Correctness of positive predictions. This avoids False Positives.
  Recall - Capturing all actual positives. This avoids False Negatives

In medical diagnosis, the more dangerous error is usually a false negative—missing a condition that actually exists.

83.F1-Score is used to balance precision and recall and is 
best used for imbalanced data sets.
               summary
A. Balanced Datasets : Use accuracy
B. Imbalance Dataset : Use F1-Score and AUC-ROC.
